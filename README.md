Overview
A Tkinter-based GUI application for face detection, landmark detection, facial alignment, and identity clustering. The program supports single image processing and bulk folder processing with automatic identity grouping using DBSCAN clustering.
Requirements

Python 3.10–3.12 recommended
Dependencies: pip install opencv-python numpy pillow scikit-learn

How to Run
python cits4402_project.py
